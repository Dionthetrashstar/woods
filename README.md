# woods

A minimalist emotional support chatbot made with Python.

**woods** is a small terminal-based conversational project focused on reflective and comforting dialogue.  
Built as a lightweight emotional interaction system, the program creates a quiet space where users can express thoughts and receive randomized supportive responses.

Inspired by melancholic dialogue, minimalist terminal applications, and atmospheric storytelling.

## Features

- Randomized comforting messages
- Dynamic conversational flow
- Context-sensitive responses
- Emotional and reflective tone
- Minimalist terminal interface
- Lightweight and simple to run

## Technologies

- Python 3
- Built-in `random` module
